# Location of `site-packages` directory

https://stackoverflow.com/questions/122327/how-do-i-find-the-location-of-my-python-site-packages-directory

## Use `site` module

https://stackoverflow.com/a/46071447/3449199

https://stackoverflow.com/a/12950101/3449199

```
python -c "import site; print(site.getsitepackages()[0])"
```

## Use `distutils.sysconfig` module

https://stackoverflow.com/a/122340/3449199

```
python -c "from distutils.sysconfig import get_python_lib; print(get_python_lib())"
```

## Use `sysconfig` module

https://stackoverflow.com/a/52638888/3449199

```
python -c "import sysconfig; print(sysconfig.get_paths()['purelib'])"
```

The `sysconfig` module is new in Python version 3.2.
